# Map Itinerary Helpers

Helpers del mapa para itinerarios.

## Archivos

- `itineraryConstants.js`: constantes visuales y de comportamiento.
- `itineraryGeometry.js`: calculos geometricos del itinerario.
- `itineraryIcons.js`: iconos usados en capas de itinerario.
- `itineraryStyles.js`: estilos de lineas y marcadores.

## Regla de mantenimiento

Esta carpeta no deberia manejar estado global. Debe entregar piezas consumibles por `useMapItinerary.js`.
