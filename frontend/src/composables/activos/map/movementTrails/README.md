# Movement Trail Helpers

Helpers para trazas de movimiento en el mapa.

## Archivos

- `movementTrailConstants.js`: constantes de trazas.
- `movementTrailData.js`: preparacion de puntos y segmentos.
- `movementTrailStyles.js`: estilos de capas.
- `movementTrailViewport.js`: encuadre y viewport de trazas.

## Regla de mantenimiento

Mantener la preparacion de datos separada del composable que actualiza Leaflet.
