# Report Config

Configuracion compartida de reportes.

## Archivos

- `reportBehaviorOptions.js`: opciones de comportamiento, como resolver direcciones o incluir eventos.
- `reportOutputOptions.js`: opciones de salida, como PDF, Excel y mapas.
- `reportWidgetUtils.js`: normalizacion de widgets visibles.
- `reportChartRecommendations.js`: sugerencias de graficos segun tipo de reporte.

## Tests

- `reportWidgetUtils.test.js`

## Criterio

Aqui deben vivir constantes y normalizadores pequenos que se usan en varios flujos.
