# Geofence History Components

Piezas visuales del historial de geocercas.

## Archivos

- `GeofenceHistoryControls.vue`: filtros y controles del historial.
- `GeofenceHistoryHeader.vue`: encabezado del modal o panel de historial.
- `GeofenceHistoryTable.vue`: tabla de eventos historicos de geocerca.

## Regla de mantenimiento

Esta carpeta solo organiza la presentacion del historial. La preparacion de filas vive en `composables/activos/geocercas/useGeofenceHistoryRows.js`.
