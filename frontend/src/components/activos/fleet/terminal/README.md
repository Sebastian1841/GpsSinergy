# Fleet Terminal Components

Componentes internos de la terminal de flota.

## Archivos

- `TerminalInput.vue`: entrada interactiva para comandos de terminal.

## Regla de mantenimiento

Mantener aqui solo UI de terminal. Parseo, filas y formato deben quedar en `utils/terminalUtils.js` o `utils/activos/fleetTerminalRows.js`.
