# UI Components

Componentes genericos de interfaz.

## Archivos

- `ColumnVisibilityMenu.vue`: selector de columnas visibles.
- `ConfirmDialog.vue`: dialogo de confirmacion reutilizable.
- `FloatingModalResizeHandles.vue`: manejadores visuales para redimensionar modales flotantes.

## Regla de mantenimiento

Estos componentes no deben conocer dominios como activos, usuarios o reportes. Deben funcionar con props y eventos genericos.
