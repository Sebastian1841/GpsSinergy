# Layout Components

Componentes estructurales de la aplicacion.

## Archivos

- `AppHeader.vue`: header principal con empresa, usuario y controles de vista.
- `AppSidebar.vue`: navegacion lateral.
- `HeaderPersonalAssetGroupsMenu.vue`: menu de grupos personales de activos.
- `PersonalAssetGroupModal.vue`: modal para crear o editar grupos personales.
- `WorkspaceSelector.vue`: selector de espacios de trabajo.

## Regla de mantenimiento

Layout puede coordinar navegacion y controles globales, pero no debe contener reglas internas de flota, reportes o usuarios.
