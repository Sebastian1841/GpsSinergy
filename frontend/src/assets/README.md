# Assets

Recursos visuales usados por la aplicacion.

## Carpetas

- `branding/`: imagenes de marca usadas en login, reportes y documentos.
- `map/`: iconos raster de vehiculos para marcadores del mapa.
- `styles/`: hojas CSS globales o especificas de vistas complejas.

## Regla de mantenimiento

Los assets deben importarse desde codigo cuando pasan por el bundler. Los archivos que deban servirse por URL publica deben vivir en `public/`.
