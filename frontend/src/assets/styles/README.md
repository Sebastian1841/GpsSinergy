# Style Assets

Hojas de estilo complementarias.

## Archivos

- `activos-map.css`: estilos globales necesarios para el mapa de activos, capas, tooltips y controles Leaflet relacionados.

## Regla de mantenimiento

Usar esta carpeta solo para CSS que no encaja bien dentro de un componente Vue. Estilos locales deben mantenerse en el componente correspondiente.
