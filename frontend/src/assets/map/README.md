# Map Assets

Iconos raster para representar tipos de activos en mapas.

## Archivos

- `vehicle-blue-bus.png`: icono para buses.
- `vehicle-blue-car.png`: icono para autos.
- `vehicle-blue-motorcycle.png`: icono para motocicletas.
- `vehicle-blue-pickup.png`: icono para camionetas.
- `vehicle-blue-trailer.png`: icono para remolques.
- `vehicle-blue-truck.png`: icono para camiones.
- `vehicle-blue-van.png`: icono para vans.
- `vehicle-yellow-machinery.png`: icono para maquinaria.

## Regla de mantenimiento

Los nombres deben describir color y tipo de activo para que `assetMarkerIcons.js` pueda mapearlos sin ambiguedad.
